Política de Privacidad

Última actualización: Junio 2026

Esta Política de Privacidad describe cómo se recopila, utiliza y comparte su información personal cuando descarga, instala y utiliza nuestra aplicación móvil de prueba (en adelante, la "Aplicación").

1. Aplicación de Prueba

Esta Aplicación ha sido desarrollada exclusivamente con fines de prueba, demostración y aprendizaje. No está destinada para un uso comercial continuo ni para el procesamiento de datos masivos.

2. Recopilación de Información

Le informamos que esta Aplicación no recopila, no almacena ni transmite ningún tipo de información personal de los usuarios (como nombres, correos electrónicos, números de teléfono o datos de ubicación exacta).

3. Datos Técnicos y de Rendimiento

La Aplicación podría recopilar información técnica no identificable de forma automática con el único fin de analizar errores o caídas del sistema (por ejemplo, el modelo de dispositivo, versión del sistema operativo Android y registros de fallos técnicos).

4. Servicios de Terceros

Esta Aplicación de prueba no utiliza servicios ni SDKs de terceros de publicidad, análisis de mercado ni redes sociales que puedan rastrear su actividad o recopilar sus datos personales.

5. Seguridad de los Datos

Aunque la Aplicación no maneja datos sensibles, nos comprometemos a proteger la integridad técnica de la herramienta mediante prácticas estándar de desarrollo seguro.

6. Cambios a esta Política

Nos reservamos el derecho de actualizar esta política en cualquier momento si la Aplicación añade nuevas funciones de prueba que requieran el uso de datos.
